# Puppeteer Scraper

## Prerequisites

- Node.js installed (version 16+ recommended)
- NPM or Yarn package manager

## Installation

### Check if Node.js and npm are installed:

```sh
node -v
npm -v
```

### Clone the repository or create a new project:

```sh
git clone <>
cd <>
```

### Install dependencies:

```sh
npm install
```

## Usage

### Run the scraper script:

```sh
node scraper.js
```
